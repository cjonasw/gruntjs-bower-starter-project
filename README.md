# gruntjs-bower-starter-project

Another starter project where I'm using GruntJS, Bower, Coffeescript, Stylus, Jade, Source Include (automatically include stylesheets and scripts).

## Getting started

```
npm install
bower install
grunt serve
```

Away you go.
